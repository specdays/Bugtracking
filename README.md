# 🐞 Bug Tracking - Swag Labs

Repositório destinado às práticas do curso **Bug Tracking com Planejamento de Testes**, realizado na plataforma **Qualiters Club**.

O objetivo deste projeto é demonstrar na prática:

- Planejamento de testes
- Execução de casos de teste
- Registro e gerenciamento de defeitos
- Organização de evidências

# 📌 Sobre o projeto

Durante o curso de **Bug Tracking**, foi possível compreender na prática como:

- Gerenciar defeitos em sistemas
- Registrar bugs
- Classificar severidade e prioridade
- Relacionar defeitos aos casos de teste
- Manter rastreabilidade entre testes e evidências

---

# 🧪 Sistema utilizado

Para a realização das práticas foi utilizada a aplicação **Swag Labs**, uma aplicação de e-commerce amplamente utilizada para treinamento de testes de software.

🔗 Acesse o sistema:

https://www.saucedemo.com/v1/

A aplicação permite validar diversos fluxos de usuário, como login, navegação, carrinho e manipulação de produtos.

# 🐞 Controle de Bugs

Durante a execução dos testes, os defeitos encontrados foram registrados em uma planilha de controle contendo:

- ID do bug
- Descrição do problema
- Story relacionada
- Versão do sistema
- Severidade
- Prioridade
- Responsável pela correção
